# RestWithASPNET
Rest API with ASP.NET in .NET 5.0

This project is a modern Rest API way for development in 2021!
Using:
  - Migrations (using Evolve)
  - MySQL
  - Version and endpoints
  - Structure layers
  - Value Objects (VO)
  - Content negociation
  - Custom serialization
  - Swagger
  - HATEOAS
  - CORS
  - Authenticação
  - File manager
  - Docker
  - Travis CI
